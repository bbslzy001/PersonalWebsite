并查集

目的：

  • 将两个集合合并
  • 询问两个元素是否在一个集合当中

基本原理：

  • 每个集合用一棵树来表示
  • 树根的编号就是整个集合的编号
  • 每个节点存储它的父节点，p[x]表示x的父节点

方法：

  • 判断树根：if(p[x]==x)
  • 求x的祖宗节点：while(p[x]!=x) x=p[x];
  • 合并两个集合：p[x]是x的集合编号，p[y]是y的集合编号，p[x]=y

合并集合：

#define _CRT_SECURE_NO_WARNINGS
#include <iostream>

using namespace std;

const int N = 100010;

int n, m;
int p[N];

int find(int x) //返回x的祖宗节点 + 路径压缩优化
{
	if (p[x] != x) p[x] = find(p[x]);
	return p[x];
}

int main(void)
{
	scanf("%d%d", &n, &m);

	//每个数都在一个集合中
	for (int i = 1; i <= n; ++i) p[i] = i;

	while (m--)
	{
		char op[2];
		int a, b;
		//%s遇到空格后停止，此时已将第一个字符写入op，但第二个字符(空格)仅从缓冲区中读出，并未写入op
		scanf("%s%d%d", op, &a, &b);
		/*
		op=="M"比较的是两个字符串的地址（仅string字符串可以这样比较st=="M"）
		strcmp(op,"M")：op<"M"返回负值，op="M"返回0，op>"M"返回正值
		对于char数组，一般需要添加零终止符，否则输出时会自动寻找该地址后面的第一个零终止符
		可以手动添加零终止符，op[1] = '\0';或op[1] = 0;
		也可以将其初始化为0，op = {0};或op = {};或op[2] = {'\0'};
		*/
		if (op[0] == 'M') p[find(a)] = find(b); //将其中一棵树的根接到另一棵树上
		else
		{
			//puts()函数的作用与语句printf("%s\n",s);的作用形同
			if (find(a) == find(b)) puts("Yes");
			else puts("No");
		}
	}

	return 0;
}
