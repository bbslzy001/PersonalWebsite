数组的定义与使用

Java中讲数组定义为引用数据类型

数组的动态初始化

  • 初始化后每一个元素的保存内容为其对应数据类型的默认值
  • 声明并初始化
    • 数据类型 数组名称 [] = new 数据类型 [长度]
    • 数据类型 [] 数组名称 = new 数据类型 [长度]

数组的静态初始化

  • 在数组定义的时候就为其设置好了内容
  • 简化格式：数据类型 数组名称 [] = { 数据1，数据2，数据3，...... }
  • 完整格式：数据类型 数组名称 [] = new 数据类型 [] { 数据1，数据2，数据3，...... }

查询数组长度

  • 具体见：关于Java中length、length()、size()的区别

foreach迭代输出

  • 利用foreach的结构可以直接获取数组中的每一个元素，避免了下标访问
  • 语法
    • for(数据类型 变量 : 数组 | 集合) {}
int data [] = new int [] {1,2,3,4,5}

for(int temp : data)
{
		System.out.println(temp);
}

二维数组

  • 动态初始化
    • 数据类型 数组名称 [][] = new 数据类型 [行个数][列个数]
  • 静态初始化
    • 简化格式数据类型 数组名称 [][] = {}
    • 完整格式数据类型 数组名称 [][] = new 数据类型 [][] {}
