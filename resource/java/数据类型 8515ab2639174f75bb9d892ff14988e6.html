数据类型

数据类型概述

  • 基本数据类型
    • 数值型
      • 整型：byte、short、int、long——————（默认值为0）
      • 浮点型：float、double—————————（默认值为0.0）
    • 布尔型：boolean—————————————（默认值为false）
    • 字符型：char———————————————（默认值为’\u0000’）
  • 引用数据类型（牵扯到内存关系的使用）
    • 数组、类、接口——————————————（默认值为null）

使用原则

  • 整数：int
  • 小数：double
  • 数据传输、文字编码转换：byte（二进制处理操作）
  • 中文：char（可选）
  • 描述内存或文件大小、描述表的主键列：long

整型

  • 整型常量默认为int
  • int的取值范围：-2147483648 ~ 2147483647
  • 若定义并赋值long时常量超过int，需要强制转换类型
    /*法一*/  long a = 2147483699L;
/*法二*/  long a = (long)2147483699;
  • 对于byte和short类型，如果定义并赋值时未超过int，可以自动转换类型
    byte a = 3;
short b = 20;

浮点型

  • 浮点型常量默认为double
  • 浮点型数值相乘时会出现不精确的情况（无解）

字符型

  • 字符型可以与int相互转换
  • java中的char类型可以保存汉字（java使用unicode十六进制编码）
  • 常见字符编码范围：
    • ‘A’(65) ~ ‘Z’(90)
    • ‘a’(97) ~ ‘z’(122)
    • ‘0’(48) ~ ‘9’(57)

布尔型

  • 布尔型与int型不能相互转换（不存在0代表false，非0代表true）
  • java中表示真假时只能用true和false
  • 例：
    • if(0){ }：不兼容的类型。实际为 int'，需要 'boolean’
    • if((boolean)1)：不可转换的类型；无法将 'int' 转换为 'boolean’

字符串

  • String（java中的字符串类型首字母必须大写）
  • 数据计算时，若运算符一侧为String类型，则另一侧会自动转换为String类型
