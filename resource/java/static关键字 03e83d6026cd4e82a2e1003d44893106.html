static关键字

static定义属性

  • 利用static设置公共属性
  • static属性既可以被对象调用，也可以被类名称调用
  • static属性可以在没有实例化对象的时候直接由类名称调用（static属性虽然定义在类中，但不受类实例化对象的控制）
  • 实例
    class Person
{
    private String name;
    private int age;
    static String country = "中华民国";

    public Person(String name, int age)
    {
        this.name = name;
        this.age = age;
    }

    public String getInfo()
    {
        return "姓名：" + this.name + "  年龄：" + this.age + "  国家：" + country;
    }
}

public class JavaDemo
{
    public static void main(String[] args)
    {
        System.out.println(Person.country);
        Person.country = "中华人民共和国";
        Person per = new Person("张三", 18);
        System.out.println(per.getInfo());
    }
}
    中华民国
姓名：张三  年龄：18  国家：中华人民共和国

static定义方法

  • static方法可以在没有实例化对象的时候直接由类名称调用
  • static方法只允许调用static属性或static方法
  • 非static方法可以调用static属性或static方法
  • 实例1
    class Person
{
    private String name;
    private int age;
    static String country = "中华民国";

    public Person(String name, int age)
    {
        this.name = name;
        this.age = age;
    }
    public static void setCountry(String c)
    {
        country =c;
    }
    public String getInfo()
    {
        return "姓名：" + this.name + "  年龄：" + this.age + "  国家：" + country;
    }
}

public class JavaDemo
{
    public static void main(String[] args)
    {
        Person.setCountry("中华人民共和国");
        Person per = new Person("张三", 18);
        System.out.println(per.getInfo());
    }
}
    姓名：张三  年龄：18  国家：中华人民共和国
  • 主类中的方法一般都要使用static关键字进行修饰，否则无法在为创建实例化对象时直接调用
    • 这点在讲java基础中的方法一节中有所提及：方法
  • 实例2：在主类中定义一个static方法并在主方法中调用
    public class JavaDemo
{
    public static void main(String[] args)
    {
        print();
    }

    public static void print()
    {
        System.out.println("Test");
    }
}
    Test
  • 实例3：在主类中定义一个非static方法并在主方法中调用
    public class JavaDemo
{
    public static void main(String[] args)
    {
        new JavaDemo().print();  //通过创建一个对象来进行调用
    }

    public void print()
    {
        System.out.println("Test");
    }
}

static应用

  • 实例：在没有传递title的时候实现自动命名title
    class Book
{
    private String title;
    private static int count = 0;

    public Book()
    {
        //调用单参构造方法，传递的参数为 NoTitle--count
				//count为静态变量，故可以独立于对象实现自增
        this("NoTitle--" + count++);
    }

    public Book(String title)
    {
        this.title = title;
    }

    public String getTitle()
    {
        return this.title;
    }
}

public class JavaDemo
{
    public static void main(String[] args)
    {
        System.out.println(new Book().getTitle());
        System.out.println(new Book("Java").getTitle());
        System.out.println(new Book().getTitle());
        System.out.println(new Book().getTitle());
    }
}
    NoTitle--0
