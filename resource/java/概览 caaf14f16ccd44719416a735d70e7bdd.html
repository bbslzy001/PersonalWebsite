概览

特点

  • 面向对象的编程语言
    • JDK版本的更新即是面向对象设计思想的不断完善
    • java也融合了面向过程和函数式编程
  • 有方便的内存回收处理机制
    • 存在优化问题
  • 使用引用代替了C中的指针
  • 支持多线程编程
    • 使得单位时间内处理性能得到提升
  • 提供高效的网络处理能力，基于NIO实现更高效的数据传输处理
  • 具有良好的可移植性

可移植性

  • 可移植性的实现是依靠JVM（java虚拟机）
    • 一个由软件或硬件模拟出来的计算机，JVM读取并处理经编译过的与平台无关的字节码文件
  • 运行机制
    • java是编译型与解释型的结合：
      • 编译器：javac.exe（针对JVM生成class文件）
      • 解释器：java.exe（将JVM的代码放在特定平台上运行）
    • java程序组成：
      • java源文件（.java文件）
      • 字节码文件（.class文件）
      • 机器码指令
    • 程序并非运行在操作系统上，而是运行在虚拟机
      • 需要考虑如何让内置应用程序充分发挥操作系统的性能
计算机高级编程语言类型
  • 编译型：C、C++
  • 解释型：APS、HTML

JDK

  • java开发工具包（包含JRE）

JRE

  • java运行时的环境
  • 只提供程序解释功能，不提供程序开发功能
