类与对象

面向对象三个特点

  • 封装性：内部操作对外部而言不可见
  • 继承性：在已有结构的基础上进行功能扩充
  • 多态性：类型的转换处理（在继承性的基础上扩充的概念）

面向对象程序开发步骤

  • OOA：面向对象分析
  • OOD：面向对象设计
  • OOP：面向对象编程

类与对象的概念

  • 类：描述某一类事物的共性的抽象概念
  • 对象：描述某一个具体产物
  • 概述：类是对象的抽象化，对象是类的实例化（先有类再有对象）
  • 组成：
    • 成员属性（Field）
    • 操作方法（Method）

类与对象的定义及使用

  • 实例
    class Person
{
    String name;
    int age;

    public void tell()
    {
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
    }
}

public class JavaDemo
{
    public static void main(String[] args)
    {
        Person per = new Person();
        per.name = "张三";
        per.age = 18;
        per.tell();
    }
}
    PS D:\_info--Java\project_test\src> javac JavaDemo.java
PS D:\_info--Java\project_test\src> java JavaDemo
姓名: 张三
年龄: 18
  • 创建对象
    • 法一）声明并实例化对象：类名称 对象名称 = new 类名称()
    • 法二）
      • 声明对象：类名称 对象名称 = null //初始化为默认值null
      • 实例化对象：对象名称 = new 类名称 ()
  • 使用对象操作类
    • 使用类的属性：实例化对象 . 成员属性
    • 使用类的方法：实例化对象 . 方法 ()

对象内存分析

  • 常用两种内存空间

    堆内存：保存的是对象的具体信息，在程序之中堆内存的开辟是通过new来完成的

    栈内存：保存的是一块堆内存的地址，即对象的地址（可以理解为对象名称）

对象引用

  • 相当于传递地址（类似于C++中的指针）
    • 写在主方法中
      public class JavaDemo
{
    public static void main(String[] args)
    {
        Person per = new Person();
        per.name = "张三";
        per.age = 18;
        Person per2 = per;
        per2.age = 20;
        per.tell();
    }
}
      姓名: 张三
年龄: 20
    • 写在其他方法中
      public class JavaDemo
{
    public static void main(String[] args)
    {
        Person per = new Person();
        per.name = "张三";
        per.age = 18;
        change(per);  //等价于Person temp = per; temp.age = 20;
        per.tell();
    }

    public static void change(Person temp)
    {
        temp.age = 20;
    }
}
      姓名: 张三
年龄: 20

垃圾空间

  • 指的是没有任何栈内存所指向的堆内存（类似C++中无指针指向但已开辟的内存）
  • 所有垃圾空间会被GC（垃圾收集器）不定期进行回收并释放空间
  • 若垃圾过多，会导致GC性能降低，从而降低程序性能

成员属性封装

  • 利用private对属性进行封装
  • 使用外部程序访问封装属性
    • 【setXxx】设置属性（可以用来初始化属性，也可以用来修改属性）
    • 【getXxx】取得属性
  • 实例
    public class JavaDemo
{
    public static void main(String[] args)
    {
        //实例化对象
				Person per = new Person();
        per.setName("张三");
        per.setAge(-18);
				//使用对象
        per.tell();
    }
}

class Person
{
    private String name;
    private int age;

    public void tell()
    {
        System.out.println("姓名: " + name);
        System.out.println("年龄: " + age);
    }

    public void setName(String n)
    {
        name = n;
    }

    public void setAge(int a)
    {
        if(a>0) age = a;
    }

    public String getName()
    {
        return name;
    }

    public int getAge()
    {
        return age;
    }
}
    姓名: 张三
年龄: 0

构造方法定义

  • 作用：实现实例化对象的属性初始化处理
  • 要求：
    • 构造方法名称必须与类名称保持一致
    • 构造方法不允许设置任何的返回值类型，即：无返回值定义
    • 在使用关键字new实例化对象时使用
  • 普通方法与构造方法的区别
    • 构造方法是在类对象实例化时调用的，普通方法是在类对象实例化产生之后调用的
  • 实例
    public class JavaDemo
{
    public static void main(String[] args)
    {
        Person per = new Person("张三", 18);
        per.tell();
    }
}

class Person
{
    private String name;
    private int age;

    public Person(String n, int a)
    {
        name = n;
        age = a;
    }

    public void tell()
    {
        System.out.println("姓名: " + name);
        System.out.println("年龄: " + age);
    }
}
    姓名: 张三
年龄: 18

构造方法重载

  • 只需要考虑参数类型及数量
  • 定义多个构造方法时，一般按照参数数量进行升序/降序排列
  • 实例

{
    public static void main(String[] args)
    {
        Person per = new Person();
        per.tell();
    }
}

匿名对象

  • 没有任何引用名称的对象，使用一次后就会变成垃圾内存，被GC回收
  • 实例
    public class JavaDemo
{
    public static void main(String[] args)
    {
        new Person("张三",18).tell();
    }
}
